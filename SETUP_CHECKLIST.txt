BEFORE YOU START:
─────────────────
□ Have your laptop with Windows OS
□ Have internet connection for downloads
□ Have USB cable for Android device (optional)
□ Have about 20-30 minutes for complete setup


PREREQUISITE INSTALLATION (5-10 minutes):
──────────────────────────────────────────

□ 1. Install Node.js
    └─ Download: https://nodejs.org/ (Latest LTS)
    └─ Run installer (next, next, finish)
    └─ Verify: Open cmd → type 'node --version'
    
□ 2. Install Android SDK
    └─ Download: https://developer.android.com/studio
    └─ Run installer
    └─ Launch Android Studio
    └─ Tools → SDK Manager → Install:
       - Android SDK Platform 30
       - Build Tools 30-32 (don't need 36+, app will find it)
       - Android Emulator (optional for testing)
    └─ Note SDK installation path (usually C:\Users\[You]\AppData\Local\Android\Sdk)

□ 3. Install Java JDK
    └─ Download: https://www.oracle.com/java/technologies/downloads/
    └─ Choose Windows x64 installer
    └─ Run installer (next, next, finish)
    └─ Note installation path (usually C:\Program Files\Java\jdk-XX.X.X)

□ 4. Install Git (Optional but recommended)
    └─ Download: https://git-scm.com/download/win
    └─ Run installer (use all defaults)


ENVIRONMENT VARIABLES SETUP (3-5 minutes):
───────────────────────────────────────────

□ 5. Open Environment Variables
    └─ Right-click 'This PC' → Properties
    └─ Click 'Advanced system settings'
    └─ Click 'Environment Variables' button
    
□ 6. Add ANDROID_SDK_ROOT
    └─ Click 'New' under User variables
    └─ Variable name: ANDROID_SDK_ROOT
    └─ Variable value: C:\Users\[YOUR_USERNAME]\AppData\Local\Android\Sdk
    └─ Click OK

□ 7. Add JAVA_HOME
    └─ Click 'New' under User variables
    └─ Variable name: JAVA_HOME
    └─ Variable value: C:\Program Files\Java\jdk-XX.X.X (or wherever you installed)
    └─ Click OK

□ 8. Edit PATH (optional but helpful)
    └─ Select 'Path' under User variables → Edit
    └─ Click 'New'
    └─ Add: C:\Users\[YOUR_USERNAME]\AppData\Local\Android\Sdk\platform-tools
    └─ Click OK


PROJECT SETUP (1-2 minutes):
────────────────────────────

□ 9. Copy project files to new laptop
    └─ Transfer the blackboxApp folder (via USB, cloud, or git clone)
    └─ Save to: C:\Users\[YOUR_USERNAME]\Documents\projects\blackboxApp

□ 10. Connect Android Device (Optional)
     └─ Connect device via USB
     └─ On device: Settings → About Phone → tap Build Number 7 times
     └─ Go to: Settings → Developer Options → Enable USB Debugging
     └─ Allow USB Debugging permission on device


THE ONE-CLICK BUILD (1 minute):
───────────────────────────────

□ 11. Open Command Prompt
     └─ Press Windows Key + R
     └─ Type: cmd
     └─ Press Enter

□ 12. Navigate to Project
     └─ Type: cd C:\Users\[YOUR_USERNAME]\Documents\projects\blackboxApp
     └─ Press Enter

□ 13. Run the Setup Script
     └─ Type: setup-and-build-android.bat
     └─ Press Enter
     └─ Watch the magic happen! (takes 5-15 minutes first time)


WHAT HAPPENS NEXT:
──────────────────

□ Script auto-runs and:
   ✓ Installs npm packages
   ✓ Fixes Android build issues
   ✓ Sets up D8 compiler
   ✓ Configures gradle
   ✓ Builds the Android app
   ✓ Deploys to connected device or emulator

□ If device is connected:
   ✓ APK will automatically install
   ✓ App will launch
   ✓ You're done! 🎉

□ If no device:
   ✓ Connect device when ready
   ✓ Or start Android emulator
   ✓ Run script again


VERIFICATION:
─────────────

After successful build, check:

□ APK built: Look for "BUILD SUCCESSFUL" in output
□ App deployed: "app-debug.apk" mentioned in output
□ Device shows app: Check your phone/emulator
□ App launches: BlackBoxApp should open on device


TROUBLESHOOTING QUICK FIXES:
────────────────────────────

If something fails:

□ "Node.js not found"
   → Close cmd, restart it, try again
   → Verify Node installed: node --version

□ "Android SDK not found"
   → Check ANDROID_SDK_ROOT is set correctly
   → Open Settings → Verify environment variable
   → Restart cmd after setting

□ "Java not found"
   → Check JAVA_HOME is set correctly
   → Path should be: C:\Program Files\Java\jdk-XX.X.X

□ "No device found"
   → Connect device + enable USB Debugging
   → Or start Android emulator first
   → Run setup script again

□ "Build tools missing"
   → Script handles this automatically
   → If still fails: Run script again
   → Or use Android SDK Manager to install build-tools
